# 雨夜书屋原始音频素材库
# Rainy Study Raw Audio Assets Library

本目录包含雨夜书屋应用使用的所有原始音频素材。

## 目录结构

```
raw_assets/
├── rain/           # 雨声音频文件
├── fire/           # 火声音频文件  
├── wind/           # 风声音频文件
├── thunder/        # 雷声音频文件
├── bird/           # 鸟鸣音频文件
└── wave/           # 海浪音频文件
```

## 音频文件规范

### 支持格式
- **.wav**: 无损格式，推荐用于主要音频
- **.mp3**: 压缩格式，用于大型文件
- **.flac**: 无损压缩格式

### 文件命名规范
- 使用英文小写字母和下划线
- 格式: `[类型]_[描述]_[时长].wav`
- 示例: `rain_gentle_30min.wav`, `fire_crackling_15min.wav`

### 音频质量要求
- **采样率**: 44.1kHz 或 48kHz
- **位深度**: 16-bit 或 24-bit
- **声道**: 立体声 (Stereo) 或单声道 (Mono)

## 音频参数配置

根据项目要求，当前配置的音频参数：

| 音频类型 | 默认音量 | 滤波设置 | 用途 |
|---------|---------|----------|------|
| 雨声 | -18dB | 低通滤波 1500Hz | 背景氛围音 |
| 火声 | -12dB | 低通滤波 1500Hz | 温暖陪伴音 |

## 使用说明

1. 将音频文件放入对应的类型目录
2. 确保文件命名符合规范
3. 在 `healing_mix.py` 脚本中引用文件名
4. 音频处理脚本会自动应用预设参数

## 注意事项

- 保持音频文件的质量，避免过度压缩
- 循环音频应确保无缝循环
- 长时间音频建议分段处理以优化内存使用
