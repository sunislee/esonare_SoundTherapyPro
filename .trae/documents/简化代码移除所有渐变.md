# 简化代码 - 移除所有渐变和光晕效果

## 问题
1. bubblingEventTypes 错误可能来自Metro缓存
2. 环形布局半径可能仍导致按钮被裁剪

## 修复方案
1. **移除所有光晕和渐变**：
   - 删除 auraBackground 和 auraCircle 样式
   - 使用纯黑背景 backgroundColor: '#000000'

2. **缩小环形半径**：
   - 将半径从 width * 0.35 减小到 width * 0.28
   - 确保所有 7 个按钮在屏幕内

3. **优化坐标计算**：
   - 按钮位置：left 在 0 到 width - 80 之间
   - 按钮位置：top 在 0 到 height - 80 之间

4. **简化层级结构**：
   - 背景层：纯黑
   - 文字层：pointerEvents="none"
   - 按钮层：zIndex 最高
