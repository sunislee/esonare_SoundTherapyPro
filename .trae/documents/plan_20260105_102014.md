# 视觉打磨计划

## 1. 优化背景光晕 (Soft Aura)

### 问题分析
- 当前使用单一颜色View，边缘有明显界限
- 不透明度较高，缺乏"微弱的氤氲感"

### 解决方案
- **多层叠加效果**：使用3-4个同心圆形View叠加，每层具有不同的透明度和大小
- **平滑淡出**：从中心到边缘透明度逐渐降低，最外层几乎透明
- **降低不透明度**：将整体不透明度降低30%，当前水平为0.6，调整后约为0.42

### 技术实现
```javascript
// 多层背景光晕
<Animated.View style={[
  styles.auraLayer1,
  { opacity: auraOpacity.interpolate({ inputRange: [0, 1], outputRange: [0, 0.42] }) }
]} />
<Animated.View style={[
  styles.auraLayer2,
  { opacity: auraOpacity.interpolate({ inputRange: [0, 1], outputRange: [0, 0.3] }) }
]} />
<Animated.View style={[
  styles.auraLayer3,
  { opacity: auraOpacity.interpolate({ inputRange: [0, 1], outputRange: [0, 0.2] }) }
]} />
<Animated.View style={[
  styles.auraLayer4,
  { opacity: auraOpacity.interpolate({ inputRange: [0, 1], outputRange: [0, 0.1] }) }
]} />
```

## 2. 实现流光过渡 (Flowing Transition)

### 问题分析
- 当前过渡时间1000ms，使用默认缓动
- 颜色变化缺乏流畅感

### 解决方案
- **延长过渡时间**：从1000ms延长至1200ms
- **使用自定义缓动**：`Easing.bezier(0.4, 0, 0.2, 1)`模拟呼吸感
- **颜色插值**：使用`Animated.interpolate`实现RGB颜色平滑过渡
- **禁用nativeDriver**：使用`useNativeDriver: false`支持颜色插值

### 技术实现
```javascript
// 流光过渡动画
Animated.timing(auraColorAnim, {
  toValue: targetColorIndex,
  duration: 1200,
  easing: Easing.bezier(0.4, 0, 0.2, 1),
  useNativeDriver: false,
}).start();

// 颜色插值
const backgroundColor = auraColorAnim.interpolate({
  inputRange: [0, 1, 2, 3, 4, 5, 6],
  outputRange: [
    scenes[0].glowColor,
    scenes[1].glowColor,
    scenes[2].glowColor,
    scenes[3].glowColor,
    scenes[4].glowColor,
    scenes[5].glowColor,
    scenes[6].glowColor,
  ],
});
```

## 3. 按钮高级质感

### 问题分析
- 当前边框宽度固定为0.5
- 未激活态文字透明度较高
- 缺乏动态描边效果

### 解决方案
- **动态描边**：激活时边框宽度从0.5变为1.5
- **外发光效果**：激活时添加外发光
- **未激活态优化**：文字透明度降至0.3

### 技术实现
```javascript
// 动态边框宽度
const borderWidth = isActive ? 1.5 : 0.5;

// 未激活态文字透明度
const textOpacity = isActive ? 1 : 0.3;

// 外发光效果
const activeGlow = {
  shadowColor: scene.color,
  shadowOffset: { width: 0, height: 0 },
  shadowOpacity: 0.5,
  shadowRadius: 10,
  elevation: 10,
};
```

## 4. 顶部栏沉浸

### 问题分析
- 确保标题栏和状态栏完全融合

### 解决方案
- **检查布局**：确保StatusBar设置正确
- **移除界限**：确保没有不必要的背景色或边框
- **调整内边距**：确保内容不被状态栏遮挡

### 技术实现
```javascript
// 确保StatusBar设置正确
<StatusBar barStyle="light-content" backgroundColor="transparent" translucent />

// 调整标题容器位置
const titleContainerStyle = {
  position: 'absolute',
  top: Platform.OS === 'android' ? StatusBar.currentHeight : 0,
  left: 0,
  right: 0,
  alignItems: 'center',
  zIndex: 999,
};
```
