## 任务分析
用户要求彻底重构Resonare项目主界面，从简陋的纯色背景+文字圆圈升级为具有"深邃、沉浸、治愈感"的现代设计风格。

## 实现步骤

### 1. 安装依赖
- 安装 react-native-linear-gradient 实现渐变背景
- 确保项目能正常编译

### 2. 背景与氛围设计
- 使用 LinearGradient 实现深藏青(#1A1C20)到暗紫(#0F1014)的渐变
- 添加缓慢流动的动画效果
- 背景填满整个屏幕，包括状态栏

### 3. 毛玻璃按钮组件
- 实现 Glassmorphism 效果：
  - 背景色 rgba(255, 255, 255, 0.05)
  - 边框 0.5px 半透明白色
- 动态激活态效果：
  - 外发光（Shadow/Glow）匹配场景颜色
  - 呼吸动画（1.0-1.05倍缩放循环）

### 4. 文字与排版
- 标题 "RESONARE"：全大写，letter-spacing: 8，字体纤细
- 状态标签："Healing: [当前场景]"，半透明小字

### 5. 交互细节
- 平滑过渡动画（500ms渐变）
- 音量控制条（毛玻璃风格，极简设计）
- 场景切换时的背景色过渡

## 关键技术点
- react-native-linear-gradient
- React Native Animated API
- 毛玻璃效果（backdrop-filter模拟）
- 场景颜色映射

## 预期结果
- 界面具有深邃、沉浸、治愈感
- 毛玻璃按钮效果精致
- 动画流畅自然
- 保持现有音频播放逻辑功能
