## "清晨河流"场景音量吓人的问题修复计划

### 问题诊断
当前 switchScene 逻辑：
```javascript
else {
  await TrackPlayer.setVolume(volume); // 直接设置到 100%
}
await TrackPlayer.play(); // 声音突然变大
```

### 修复方案

#### 1. 静音启动 + 异步淡入
为 "清晨河流" 场景添加特殊处理：
```javascript
// 如果是清晨河流，静音启动 + 异步淡入
else if (targetScene.id === 'morning_river') {
  await TrackPlayer.setVolume(0); // 先设为0
  // 异步执行淡入（0.1 -> 0.4，15秒）
  setTimeout(() => fadeInForMorningRiver(), 50);
}
```

#### 2. 新增淡入函数
```javascript
const fadeInForMorningRiver = async () => {
  const startVolume = 0.1; // 从10%开始
  const targetVolume = 0.4; // 增加到40%
  const duration = 15000; // 15秒
  const interval = 100;
  const steps = duration / interval;
  const volumeStep = (targetVolume - startVolume) / steps;
  
  let curVol = startVolume;
  await TrackPlayer.setVolume(startVolume);
  
  fadeRef.current = setInterval(async () => {
    curVol += volumeStep;
    if (curVol >= targetVolume) {
      curVol = targetVolume;
      clearInterval(fadeRef.current);
      fadeRef.current = null;
    }
    await TrackPlayer.setVolume(curVol);
  }, interval);
};
```

#### 3. 修改位置
- switchScene 函数（约第 541-543 行）
- 添加 fadeInForMorningRiver 函数（约第 692 行后）

- ✅ 15秒完成淡入，之后稳定在 40%
