## 修改 SCENES 数组中的音频文件引用

### 1. 修改梦境漂流场景
将第 13 行的引用从：
```javascript
require('./assets/audio/tracks/Dream Drift.mp3')
```
改为：
```javascript
require('./assets/audio/tracks/dream_drift.mp3')
```

### 2. 修改流动的宁静场景
将第 14 行的引用从：
```javascript
require('./assets/audio/tracks/Liquid Peace.m4a')
```
改为：
```javascript
require('./assets/audio/tracks/liquid_peace.m4a')
```

### 3. Metro Server 自动重载
修改完成后，Metro Server 会自动检测到文件变化并重新加载应用。如果没有自动重载，可以手动重启：
- 在运行 Metro 的终端按 `Ctrl + C` 停止
- 重新运行 `npm start`
