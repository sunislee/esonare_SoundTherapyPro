## 修改计划

### 任务 1：修改全局 Gradle 配置
- **文件**：`android/build.gradle`
- **位置**：文件末尾（第 48 行之后）
- **修改内容**：添加以下块，强制所有子项目同步 Kotlin 设置并抑制类型检查报错：
```gradle
allprojects { 
     configurations.all { 
         resolutionStrategy { 
             force "org.jetbrains.kotlin:kotlin-stdlib:$kotlinVersion" 
             force "org.jetbrains.kotlin:kotlin-reflect:$kotlinVersion" 
         } 
     } 
     subprojects { 
         afterEvaluate { project -> 
             if (project.hasProperty('android')) { 
                 project.android { 
                     kotlinOptions { 
                         freeCompilerArgs += ["-Xjvm-default=all", "-Xsuppress-version-warnings"] 
                     } 
                 } 
             } 
         } 
     } 
}
```
- **目的**：确保所有子项目使用相同的 Kotlin 版本，并抑制类型检查报错
