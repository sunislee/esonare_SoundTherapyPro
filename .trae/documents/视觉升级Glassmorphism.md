# 视觉升级 - Glassmorphism风格

## 升级内容

1. **按钮Glassmorphism效果**：

   * 背景 rgba(255, 255, 255, 0.05)

   * 非选中边框 0.3px rgba(255,255,255,0.15)

   * 选中按钮呼吸动画 scale 1.0-1.08，周期3000ms

2. **环境光晕动画**：

   * auraGlow 透明度从0渐变到0.2

   * 多层rgba叠加模拟向外扩散效果

3. **文字层级优化**：

   * RESONARE opacity: 0.9，颜色 #F0F0F0

   * 背后增加黑色阴影

4. **交互细节**：

   * 按下缩放到0.92，松开恢复
