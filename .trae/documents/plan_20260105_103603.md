# 修复计划

## 1. 修复崩溃问题 (Style Property Error)

### 问题分析
- 可能存在未使用的动画属性
- 需要确保所有非 opacity/transform 的动画属性都使用 `useNativeDriver: false`

### 解决方案
- 移除未使用的 `glowAnim` 动画
- 检查所有 Animated.timing 和 Animated.spring 调用
- 确保只有 opacity 和 transform 动画使用 `useNativeDriver: true`

### 技术实现
```javascript
// 移除未使用的 glowAnim
// const glowAnim = useRef(new Animated.Value(0)).current;

// 检查所有动画配置
Animated.timing(breathAnim, {
  toValue: 1,
  duration: 2000,
  easing: Easing.inOut(Easing.ease),
  useNativeDriver: true, // 正确，opacity 是允许的
});
```

## 2. 修复视觉遮挡 (Layering & Transparency)

### 问题分析
- 标题容器未使用 `StyleSheet.absoluteFillObject`
- 背景光晕透明度可能仍然太高
- 背景光晕可能使用了实心颜色

### 解决方案
- 将标题放入 `StyleSheet.absoluteFillObject` 容器
- 设置 `pointerEvents="none"` 和最高 zIndex
- 将背景光晕透明度降低到 0.05
- 确保所有 View 除了根容器和按钮，背景都是 transparent

### 技术实现
```javascript
// 标题容器样式
const titleContainerStyle = {
  ...StyleSheet.absoluteFillObject,
  alignItems: 'center',
  justifyContent: 'center',
  zIndex: 999,
  pointerEvents: 'none',
};

// 背景光晕透明度
const auraOpacity = useRef(new Animated.Value(0)).current;
// toValue: 0.05
```

## 3. 修正环形布局 (Circular Layout)

### 问题分析
- 按钮圆环半径可能不够大
- 按钮可能包围并遮挡中间文字

### 解决方案
- 增大按钮圆环半径（在当前基础上增加 30px 以上）
- 将按钮整体向下平移

### 技术实现
```javascript
// 增大半径并向下平移
const radius = width * 0.65 + 30;
const y = Math.sin(angle) * radius + 50; // 向下平移 50px
```

## 4. 简化样式 (Cleanup)

### 问题分析
- 可能存在导致"白条"或"黑块"的全局 backgroundColor
- 需要确保样式简洁干净

### 解决方案
- 检查并移除所有不必要的 backgroundColor
- 确保除了根容器和按钮，其他 View 背景都是 transparent

### 技术实现
```javascript
// 检查所有 View 样式
<View style={{ backgroundColor: 'transparent' }}>
  {/* 内容 */}
</View>
```
