## App.js 优化方案：高级感 UI

### 1. 添加呼吸动画
- 使用 Animated 库
- 当音频播放（currentId === s.id）时，对应的圆圈产生持续的呼吸缩放效果
- 使用 Animated.timing 或 Animated.spring 实现平滑动画

### 2. 优化背景渐变
- 点击不同场景时，背景色不再是纯黑
- 平滑过渡到该场景定义的 color 的深色渐变版
- 使用 LinearGradient 实现背景渐变

### 3. 提升质感
- 给圆圈添加细微的内阴影（shadowColor, shadowOpacity, shadowRadius）
- 文字加一点字间距（letterSpacing）
- 让整体看起来更有冥想 App 的高级感

### 4. 实现细节
- 优化圆圈样式（borderWidth, borderColor）
- 添加按压反馈（opacity 变化）
- 优化文字样式（fontSize, fontWeight）

完成后执行 `npm run android` 测试效果
