# App 调试计划

## 1. 环境检查
- 检查 Node.js 版本（需要 >=20）
- 检查依赖是否已安装（node_modules）
- 检查设备连接状态（Android/iOS 真机）

## 2. 清理和重置
- 清理 Metro bundler 缓存
- 清理 Android/iOS 构建缓存
- 重新安装依赖（如需要）

## 3. 尝试运行
- 启动 Metro bundler
- 运行 Android 或 iOS 应用
- 查看控制台输出和错误日志

## 4. 问题诊断
- 根据错误信息进行针对性修复
- 可能需要：
  - 修复依赖冲突
  - 更新配置文件
  - 修复代码错误
  - 处理权限问题

## 5. 验证修复
- 确保应用能在真机上正常运行
- 测试基本功能

**注意**：根据项目规则，所有调试操作必须基于 Android/iOS 真机，严禁使用模拟器。
