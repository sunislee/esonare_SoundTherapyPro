## 重构计划：将音频效果整合为平级场景

### 1. 修改场景相关常量
- **更新 sceneFrequencies**：添加 Alpha 波、粉红噪音、双耳节拍的频率
- **更新 sceneSoundModes**：添加这三个场景的声音模式
- **更新 sceneBackgrounds**：为这三个场景分配背景图片
- **更新 sceneParticles**：为这三个场景分配粒子效果（如果需要）

### 2. 移除旧的音频效果状态
- 删除 `alphaWaveEnabled`、`pinkNoiseEnabled`、`binauralBeatEnabled` 状态变量
- 删除 `handleAudioEffectToggle` 函数
- 删除音频效果开关面板 UI 组件

### 3. 更新 changeScene 函数
- 添加音频互斥逻辑：切换场景时先停止当前音频
- 根据场景类型设置正确的音频参数
- 更新状态栏显示文本

### 4. 修改 UI 布局
- 修改 `sceneContainer` 样式，支持显示 7 个场景（分两行排列）
- 移除 `effectsContainer` 组件
- 更新状态文本显示逻辑，根据场景显示不同的状态

### 5. 确保音频互斥
- 确保每次调用 `changeScene` 时，SuperpoweredModule 收到唯一的场景指令
- 实现音频停止和启动的逻辑，确保声音不会叠加

4. **修改 changeScene 函数**：
   ```javascript
   const changeScene = (scene) => {
     if (!SuperpoweredModule) {
       Alert.alert("错误", "原生模块未链接！");
       return;
     }

     // 获取当前场景的参数
     const frequency = sceneFrequencies[scene];
     const soundMode = sceneSoundModes[scene];
     
     // 先停止当前音频（如果正在播放）
     if (isGenerating) {
       SuperpoweredModule.toggleAudio();
     }
     
     // 设置频率和声音模式
     if (soundMode === 'alpha') {
       // Alpha 波特殊处理
       SuperpoweredModule.setAudioEffect('alpha', true);
     } else {
       // 其他场景，先关闭 Alpha 波
       SuperpoweredModule.setAudioEffect('alpha', false);
       SuperpoweredModule.setFrequency(frequency);
       SuperpoweredModule.setSoundMode(soundMode);
     }
     
     // 重启音频
     SuperpoweredModule.toggleAudio();
     
     // 更新状态
     setCurrentScene(scene);
     setCurrentFrequency(frequency);
     setCurrentSoundMode(soundMode);
     
     // 更新状态栏文本
     if (scene === 'Alpha 波') {
       setStatus('当前模式：Alpha 疗愈');
     } else {
       setStatus(`Current Scene: ${scene} | ${soundMode === 'binaural' ? '双耳节拍' : '粉红噪音'}`);
     }
   };
   ```

5. **修改 UI 布局**：
   - 移除 `effectsContainer` 组件
   - 修改 `sceneContainer` 样式，支持分两行排列
