## "清晨河流"场景音量"黄金曲线"更新计划

### 修改内容

#### 1. 两阶段淡入曲线
```javascript
const fadeInForMorningRiver = async () => {
  // 第一阶段：前3秒，0.25 -> 0.35
  const phase1Start = 0.25;
  const phase1End = 0.35;
  const phase1Duration = 3000;
  const phase1Interval = 50; // 更平滑的步进
  const phase1Steps = phase1Duration / phase1Interval;
  const phase1Step = (phase1End - phase1Start) / phase1Steps;
  
  // 第二阶段：后12秒，0.35 -> 0.5
  const phase2Start = 0.35;
  const phase2End = 0.5;
  const phase2Duration = 12000;
  const phase2Interval = 100;
  const phase2Steps = phase2Duration / phase2Interval;
  const phase2Step = (phase2End - phase2Start) / phase2Steps;
  
  // 先静音
  await TrackPlayer.setVolume(0);
  
  // 立即跳到 0.25 开始播放
  let curVol = phase1Start;
  await TrackPlayer.setVolume(curVol);
  
  // 第一阶段：3秒快速爬升到 0.35
  for (let i = 0; i < phase1Steps; i++) {
    await new Promise(r => setTimeout(r, phase1Interval));
    curVol += phase1Step;
    await TrackPlayer.setVolume(Math.min(curVol, phase1End));
  }
  
  // 第二阶段：12秒平滑爬升到 0.5
  for (let i = 0; i < phase2Steps; i++) {
    await new Promise(r => setTimeout(r, phase2Interval));
    curVol += phase2Step;
    await TrackPlayer.setVolume(Math.min(curVol, phase2End));
  }
};
```

#### 2. switchScene 中的调用保持不变
```javascript
else if (targetScene.id === 'morning_river') {
  await TrackPlayer.setVolume(0); // 先设为0静音
```

### 音量曲线参数
- **0ms**: 静音 (0)
- **50ms**: 立即跳到 0.25 (可听见的起始点)
- **0-3秒**: 0.25 -> 0.35 (建立听觉存在感)
- **3-15秒**: 0.35 -> 0.5 (达到疗愈深度)
