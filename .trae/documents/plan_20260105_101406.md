# 电影级视觉升级计划

## 1. 彻底改造选中效果 (The Aura Effect)

### 问题分析
- 当前氛围光为实心圆，透明度高，遮挡标题
- 颜色切换无过渡动画

### 解决方案
- **移除实心圆氛围光**：删除当前的`ambientGlow`实心圆
- **创建巨大渐变光晕**：
  - 使用`Animated.View`创建全屏或超大尺寸的光晕
  - 通过`shadowColor`、`shadowRadius`和`shadowOpacity`实现高模糊效果
  - 中心镂空：将光晕置于按钮容器下方，标题置于最顶层
- **平滑颜色过渡**：
  - 添加1000ms的颜色渐变动画
  - 使用`Animated.timing`实现RGB颜色插值

### 技术实现
```javascript
// 全屏渐变光晕
<Animated.View 
  style={[
    styles.auraGlow,
    {
      backgroundColor: currentScene?.glowColor || 'transparent',
      opacity: auraOpacity,
      transform: [{ scale: auraScale }]
    }
  ]} 
/>
```

## 2. 引入呼吸感动画 (Breathing Animation)

### 问题分析
- 当前选中按钮无呼吸效果
- 动画不够生动

### 解决方案
- **为选中按钮添加呼吸动画**：
  - 在`SceneButton`组件中添加`breathAnim`
  - 控制`opacity`在0.3-0.8之间往复
  - 控制`scale`在1.0-1.08之间往复
  - 动画周期约4秒

### 技术实现
```javascript
// 呼吸动画
useEffect(() => {
  if (isActive) {
    Animated.loop(
      Animated.sequence([
        Animated.timing(breathAnim, {
          toValue: 1,
          duration: 2000,
          easing: Easing.inOut(Easing.ease),
          useNativeDriver: true,
        }),
        Animated.timing(breathAnim, {
          toValue: 0,
          duration: 2000,
          easing: Easing.inOut(Easing.ease),
          useNativeDriver: true,
        }),
      ])
    ).start();
  }
}, [isActive]);
```

## 3. 优化文字层级 (Hierarchy)

### 问题分析
- 标题zIndex不够高
- 字体粗细和间距不符合要求
- 选中文字无发光效果

### 解决方案
- **提升标题层级**：将`titleContainer`的zIndex设为999
- **更新标题样式**：
  - `fontWeight: '100'`
  - `letterSpacing: 12`
- **选中文字效果**：
  - 稍微加粗（`fontWeight: '600'`）
  - 添加同色系微弱发光（通过`textShadowColor`实现）

### 技术实现
```javascript
// 标题样式
title: {
  fontSize: 24,
  color: '#FFFFFF',
  fontWeight: '100',
  letterSpacing: 12,
  zIndex: 999,
  // ...其他样式
}

// 选中文字发光效果
buttonText: {
  // ...其他样式
  textShadowColor: isActive ? scene.color : 'transparent',
  textShadowOffset: { width: 0, height: 0 },
  textShadowRadius: isActive ? 5 : 0,
  fontWeight: isActive ? '600' : '500',
}
```

## 4. 按钮质感提升

### 问题分析
- 非选中按钮边框透明度较高
- 按钮质感不够精致

### 解决方案
- **降低边框透明度**：将非选中状态的`borderColor`透明度从0.2降至0.1
- **保持选中状态的边框效果**：
  - 选中时边框颜色与场景颜色一致
  - 增强选中状态的视觉反馈

### 技术实现
```javascript
// 按钮边框样式
sceneButton: {
  // ...其他样式
  borderColor: isActive ? scene.color : 'rgba(255, 255, 255, 0.1)',
  // ...其他样式
}
```
