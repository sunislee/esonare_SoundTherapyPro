# 极简代码重写 - 确保无任何渐变库

## 问题
mobile_app 中发现 gradient 残留，但 RainyStudy 中没有 LinearGradient

## 重写方案
1. **使用极简坐标公式**：
   - centerX = ScreenWidth / 2
   - centerY = ScreenHeight / 2  
   - radius = ScreenWidth * 0.35
   - angle = (index * 2 * Math.PI) / 6

2. **修复交互遮挡**：
   - 文字层 pointerEvents="none", zIndex: 100
   - 按钮层 zIndex: 999

3. **根治崩溃**：
   - 根容器只使用 <View style={{flex: 1, backgroundColor: '#000'}} />

4. **6个按钮**：
   - 使用6个场景（去掉binaural双耳节拍）
   - 确保按钮完整显示在屏幕内
