# 雨夜书屋 (Rainy Study) - 音频参数配置

## 项目概述

雨夜书屋是一个跨平台沉浸式音频体验应用，结合现代音频处理技术和自然音频素材，为用户提供宁静治愈的听觉体验。

## 技术架构

### 前端框架
- **React Native**: 跨平台移动应用开发框架
- 支持 iOS 和 Android 双平台

### 音频引擎
- **Superpowered SDK**: 专业级C++音频处理库
- 底层高性能音频混音和处理
- 低延迟音频播放

### 开发工具
- **Python音频处理**: 使用 pydub 和 numpy 进行音频预处理
- **C++音频引擎**: 核心音频混音逻辑

## 音频参数配置

### 当前音频参数设置

| 音频类型 | 音量设置 | 滤波设置 | 描述 |
|---------|---------|----------|------|
| 雨声 | **-18 dB** | 低通滤波 1500Hz | 营造宁静的背景氛围 |
| 火声 | **-12 dB** | 低通滤波 1500Hz | 提供温暖的陪伴感 |
| 主音量 | 可调节 | - | 全局音频控制 |

### 参数说明

#### 音量控制 (dB值)
- **dB (分贝)**: 音频工程中的标准音量单位
- **负值**: 表示相对于最大音量的衰减
- **雨声 -18dB**: 相对较低，作为背景音不抢夺注意力
- **火声 -12dB**: 适中音量，提供明显的温暖感

#### 低通滤波器
- **截止频率**: 1500Hz
- **作用**: 过滤掉高频成分，使音频更加柔和
- **效果**: 减少尖锐感，增加舒适度

### 音频文件结构

```
audio_engine/
├── scripts/
│   └── healing_mix.py          # 音频混音脚本
├── raw_assets/                 # 原始音频素材库
│   ├── rain/                   # 雨声音频文件
│   ├── fire/                   # 火声音频文件
│   └── ...                     # 其他自然音频
└── requirements.txt            # Python依赖包

native_modules/
├── AudioEngine.h               # 音频引擎头文件
└── AudioEngine.cpp             # 音频引擎实现
```

## 开发环境配置

### Python环境
```bash
cd audio_engine
pip install -r requirements.txt
```

### 音频处理脚本使用
```bash
cd audio_engine/scripts
python healing_mix.py
```

### React Native环境
```bash
cd mobile_app
npm install
```
